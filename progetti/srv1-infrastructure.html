srv1 Infrastructure

L architettura invisibile dietro la massima disponibilita.The invisible architecture behind maximum availability.

La sfidaThe challenge

Creare una base infrastrutturale per hostare 30 container Docker garantendo sicurezza perimetrale, ridondanza dati e monitoraggio costante. Su hardware consumer, senza servizi enterprise.Creating an infrastructure foundation to host 30 Docker containers ensuring perimeter security, data redundancy, and constant monitoring. On consumer hardware, without enterprise services.

La soluzioneThe solution

Ho implementato una strategia di difesa in profondita: hardening UFW, networking privato via Tailscale e Headscale, pipeline di backup 3-2-1 con rsync offsite. Il monitoraggio e centralizzato su dashboard personalizzata.I implemented a defense-in-depth strategy: UFW hardening, private networking via Tailscale and Headscale, 3-2-1 backup pipeline with offsite rsync. Monitoring is centralized on a custom dashboard.

Tech Stack

Linux Ubuntu Docker UFW Tailscale Headscale rsync
