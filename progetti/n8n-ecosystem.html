n8n Ecosystem

Orchestrazione intelligente per sistemi complessi.Intelligent orchestration for complex systems.

La sfidaThe challenge

Gestire trenta workflow di automazione su un server self-hosted richiede resilienza. Ogni workflow deve essere monitorato, diagnosticato e riparato senza intervento manuale.Managing thirty automation workflows on a self-hosted server requires resilience. Each workflow must be monitored, diagnosed, and repaired without manual intervention.

La soluzioneThe solution

Ho costruito un ecosistema di 30 workflow n8n. Include un Telegram Link Router a 19 nodi che classifica via LLM i link in arrivo, un Auto-Healer che diagnostica e ripara workflow falliti, e Uptime Kuma per heartbeat e monitoraggio uptime. Tutto orchestra su un singolo server.I built an ecosystem of 30 n8n workflows. It includes a 19-node Telegram Link Router that classifies incoming links via LLM, an Auto-Healer that diagnoses and repairs failed workflows, and Uptime Kuma for heartbeat uptime monitoring. All orchestrated on a single server.
