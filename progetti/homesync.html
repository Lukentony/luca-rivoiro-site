La sfidaThe challenge

Le app di task management hanno tutte lo stesso problema: i tuoi dati stanno su server di qualcun altro. Volevo qualcosa di privato, facile da tenere in vita, che girasse sul mio server senza richiedere manutenzione continua.All task management apps have the same problem: your data lives on someone else's server. I wanted something private, easy to maintain, running on my own server without constant upkeep.

La soluzioneThe solution

Backend FastAPI asincrono con SQLite/WAL per persistenza zero-config. Frontend React senza build step (Babel standalone, JSX in-browser) che carica in 0.3 secondi. Docker multi-container con healthcheck e rate limiting. Notifiche push server-side (VAPID/Web Push, arrivano anche ad app chiusa), calendario ricorrenze, supporto multi-utente. Lo uso ogni giorno con la mia compagna e se qualcosa si rompe, lo scopro subito.Async FastAPI backend with SQLite/WAL for zero-config persistence. React frontend with no build step (Babel standalone, JSX in-browser) loading in 0.3s. Multi-container Docker with healthcheck and rate limiting. Server-side push notifications (VAPID/Web Push, delivered even when the app is closed), recurring calendar, multi-user support. I use it every day with my partner — if something breaks, I know immediately.

Tech Stack
